# Build Instructions

## Requirements
- Android Studio
- JDK 17
- Gradle 8+
- Android SDK 34
- AndroidLibV2rayLite

## How to Build
1. Clone the repository:
   git clone https://github.com/NetGoVPN-Team/NetGoVPN-Android.git
2. Open the project in Android Studio
3. Sync Gradle
4. Run the app on an emulator or device
